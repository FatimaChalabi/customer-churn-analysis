# 📊 Customer Churn Analysis Dashboard

This repository contains an end-to-end data analytics project focused on understanding customer churn and identifying key behavioral and demographic factors that affect it. The project utilizes **SQL** for data extraction and preparation, and **Microsoft Power BI** for visualization.

## 📑 Dashboard Structure

The Power BI dashboard consists of three interactive pages:

### 1. Executive Summary
Provides a high-level overview of the key business metrics:
* **Churn Rate %**: Overall percentage of churned customers.
* **Monthly Revenue**: Total monthly recurring revenue.
* **Total Customers**: Count of unique customers.
* **Total Revenue**: Cumulative revenue generated.

### 2. Demographic Analysis
Analyzes the impact of demographic characteristics on churn:
* **Churn by Gender**: Visualizes churn distribution by gender.
* **Revenue by Tenure**: Explores revenue generated across different customer tenures.
* **Churn by Senior Citizen**: Displays the churn rate for senior citizens.

### 3. Behavioral Trends
Examines customer behavior and financial metrics:
* **Total Charges vs Monthly Charges**: A scatter plot comparing total vs. monthly charges.
* **Monthly Charges Trend by Tenure**: A line chart showing the trend of monthly charges over the customer's tenure.
* **Customer Details**: A detailed table listing customer IDs and their metrics.

## 🛠️ Tools & Technologies
* **SQL**: Data extraction, data cleaning, and calculating initial metrics.
* **Microsoft Power BI**: Data modeling, interactive dashboard design, and visualizations.

## 🚀 How to View the Dashboard
1. Download the `.pbix` file from this repository.
2. Open it with **Microsoft Power BI Desktop**.
